# 药店管理系统简介 python221

## 项目概述

药店管理系统是基于Python语言和Django框架开发的一套业务处理系统，旨在提高药店内部工作效率，优化药品管理流程，为管理员、医生和患者提供便捷的服务。

## 技术栈

- 主要开发语言：Python
- Web开发框架：Django
- 数据库：MySQL

## 功能模块

### 核心功能

- 个人中心
- 公告信息管理
- 医生管理
- 患者管理
- 药品信息管理
- 开药记录管理
- 入库记录管理
- 出库记录管理
- 药品盘点管理
- 系统管理

### 系统角色

- 管理员
- 医生
- 患者

## 运行环境
